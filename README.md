# gcd-algorithm-analysis-python

## Description
Python analysis project comparing greatest common divisor algorithms with timing experiments, CSV/XLSX exports, and generated plots.

Academic context: algorithm analysis assignment for PAA (Projeto e Analise de Algoritmos), comparing brute force, Euclid, and iterative Euclid approaches.
## Tech Stack
- Python
- pandas
- matplotlib
- NumPy
- Single-language project

## Structure
- `gcd_algorithm_analysis.py`: main entry point.
- `requirements.txt`: Python dependencies for the analysis and charts.
- `resultados_mdc*.csv`: generated timing/result tables.
- `*.png`: generated charts.
- Python script contains the algorithm implementations, timings, CSV handling, and plotting logic.

## How to Run
pip install -r requirements.txt
python gcd_algorithm_analysis.py
